DealSpot Privacy Policy

Last updated: December 04, 2025

1. Overview

DealSpot is a Shop Mini that runs inside the Shopify Shop app. We help you discover products through curated sections (Trending, Recommended, Recent, and Saved) and provide filtering tools to refine your search. We rely on the Shop Minis SDK for all user interactions, storage, and checkout.

2. Data We Display

Everything you see inside DealSpot comes directly from Shopify:

  • Product catalog details, pricing, and discounts supplied by merchants.
  • Product rankings, ratings, and review data available through the Shop Minis API.
  • Merchant profiles, shop information, and fulfillment metadata owned by Shopify.
  • Your recent browsing history and saved products (stored locally via Shop Minis SDK).

3. Data We Collect

We only collect lightweight, anonymous information to keep the mini running smoothly:

  • Basic usage analytics such as which sections you visit and products you view.
  • Filter preferences you set (stored locally via Shop Minis SDK).
  • Performance data provided by the Shop Minis SDK (for example, screen load timing).
  • Crash or error details reported through Shopify's diagnostic tools.

We do not receive personal identifiers (email, address, payment info) from Shopify.

4. How We Use This Data

  • Display your personalized product recommendations and browsing history.
  • Remember your filter preferences to improve your browsing experience.
  • Measure which product discovery features work best on mobile devices.
  • Monitor reliability so that experiences stay fast and responsive.

5. Storage & Security

All data flows through Shopify’s infrastructure. We rely on the Shop Minis SDK for storage, secure communication, and authentication. We do not export or store data outside Shopify systems.

6. Data Retention

Anonymous analytics logs are kept only as long as Shopify retains them for operational purposes. If Shopify deletes your Shop account or clears activity data, DealSpot automatically loses access as well.

7. Your Choices

  • Manage data sharing preferences inside the Shop app privacy settings.
  • Clear the Shop app’s local data to remove cached DealSpot activity.
  • Stop using DealSpot at any time—no information is kept after the Shop app removes it.

8. Changes to This Policy

If we update how DealSpot handles information, we will refresh this page and revise the “Last updated” date. Keep using DealSpot only if you agree with the latest policy.

9. Questions

Need help? Open the Shop app, go to Account → Help & support, and reference the DealSpot mini. Shopify will route questions to us if needed.
