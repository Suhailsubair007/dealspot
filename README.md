# DealSpot – Shopify Shop Mini

DealSpot is a simple Shopify Shop Mini created to help users easily discover discounted products and high-rated popular picks across different stores. The mini focuses on clean design, fast browsing, and making it easy for users to find valuable deals without searching manually through multiple stores.

---

## Features

- Displays all discounted products from various stores  
- Sorts products based on highest discount  
- Shows clear discount badges (e.g., “40% OFF”, “Save ₹300”)  
- Shows original price versus current sale price  
- Includes a Mega Deals section for 50%+ discounts  
- Includes a Popular Products section based on ratings and review count  
- Optional: Store-wise deal grouping  

---

## Tech Stack

- Shopify Shop Minis SDK  
- React  
- TypeScript  

---

## Setup Instructions

1. Install the Shop Mini template:  
   ```
   npm init @shopify/shop-mini@latest
   ```
2. Install the dependecies:  
   ```
   npm install
   ```   

3. Start the development server:  
   ```
   npx shop-minis dev
   ```

3. Open the Shop app in developer mode to preview your mini.

---

## Conclusion

DealSpot is built to provide users with a quick and easy way to discover the best deals and top-rated products inside the Shop app. It is simple, lightweight, and focused entirely on user convenience. This mini can be expanded further with additional filters, categories, and store-based deal organization as needed.
